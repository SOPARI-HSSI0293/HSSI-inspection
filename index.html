⚙ Pengaturan Koneksi Google Sheets
Deploy Apps Script sebagai Web App, paste URL-nya di sini. Lihat panduan di README.
Dari URL Google Sheets: docs.google.com/spreadsheets/d/[ID ini]/edit
Struktur Google Sheets yang diperlukan:
Sheet 1: tbl_users — kolom: username, password, nama_lengkap
Sheet 2: inspection_result — kolom: ResultID, Slit_COIL_NO, ProcessNo, InspectedBy, Size_Name, InspectionTimestamp, SL_Width_Top1_Actual, Thick_Top1_Actual, Burr_Top_Actual, SL_Width_End1_Actual, Thick_End1_Actual, chk_get_coil_no
Sheet 3: process_data — kolom: ProcessNo, Customer, MotherCoil, Material, ThickNominal, WidthNominal, ThickTol, WidthTol, BurrMax
Sheet 4: slit_coil_list — kolom: ProcessNo, SlitCoilNo, SizeName, DevideNo
Mode Offline (Tanpa Google Sheets):
Jika URL belum dikonfigurasi, aplikasi menggunakan data demo lokal (localStorage). Cocok untuk testing form sebelum integrasi Sheets.
